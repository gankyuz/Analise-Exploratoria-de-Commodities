# Análise Exploratória de Commodities
**Descrição do Projeto**

Este projeto tem como objetivo realizar uma análise exploratória de dados (Exploratory Data Analysis – EDA) sobre preços históricos de diferentes commodities, abrangendo três grandes categorias: energia, metais e produtos agrícolas. A análise busca identificar padrões de comportamento, variações de preços, correlações entre produtos e tendências temporais ao longo dos anos.

O estudo foi desenvolvido a partir de um dataset disponível no Kaggle, contendo séries temporais de preços de diversas commodities.

Link: https://www.kaggle.com/datasets/richeyjay/commodity-prices-dataset

**Conjunto de Dados**

O conjunto de dados é composto por preços históricos de commodities, organizados originalmente no formato largo, em que cada produto é representado por uma coluna distinta. Entre os produtos analisados, destacam-se:
* Petróleo WTI
* Petróleo Brent
* Gás Natural
* Cobre
* Alumínio
* Milho
* Trigo
* Café
  
**Stack Utilizada**
* Python
* Pandas
* NumPy
* Matplotlib 
* Seaborn
* Jupyter Notebook

**Metodologia**

* Carregamento e exploração inicial do conjunto de dados;
* Análise descritiva dos preços das commodities;
* Comparação entre produtos de uma mesma categoria;
* Análise da variação de preços ao longo do tempo;
* Análise de correlação entre commodities;
* Comparação entre categorias distintas de produtos;
* Visualização gráfica dos resultados por meio de gráficos de linhas, mapas de calor e gráficos comparativos.


**Principais Resultados**

* Os produtos energéticos, especialmente os petróleos, apresentam maior volatilidade de preços;
* O cobre se destaca como a commodity de maior valor entre os metais;
* Produtos agrícolas como milho, trigo e café apresentam comportamentos correlacionados;
* Eventos econômicos e globais refletem diretamente nas oscilações dos preços das commodities;
* Existe correlação significativa entre determinados produtos, especialmente dentro da mesma categoria.

**Conclusões**

A análise exploratória permitiu compreender o comportamento histórico dos preços de diferentes commodities, evidenciando padrões, correlações e impactos de eventos externos no mercado. O projeto demonstra a importância da visualização de dados e da análise comparativa para a interpretação de séries temporais econômicas.
